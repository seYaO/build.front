## git命令
- 克隆  git clone git@git.17usoft.com:intervacations/tc-cruise-node-mvc.git
- 创建本地分支  git checkout -b 分支名称
- 删除本地分支  git branch -d 分支名称 （需要切换到其他分支进行操作）
- 拉取   git pull
- 合并   git merge 其他分支 （到当前分支）
