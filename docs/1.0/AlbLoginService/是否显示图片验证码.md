# 是否显示图片验证码（普通登录）

### 服务对应方法名
##### functionCode
 `isShowImg`

##### 接口请求返回示例
返回示例：
```json
{
    "code": "0000",
    "message": "成功",
    "data": {
        "result": false
    }
}
```

##### 请求参数说明
节点 | 节点描述 | 取值说明
---|---|---
acount | 帐号 | 手机号

##### 返回参数说明
节点 | 父节点 | 节点描述 | 取值说明
---|---|---|---
result | data | 是否显示 | true: 显示验证码, false: 不显示验证码
