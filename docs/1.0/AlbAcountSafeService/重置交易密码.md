# 重置交易密码

### 服务对应方法名
##### functionCode
 `resetPayPwd`

##### 接口请求返回示例
返回示例：
```json
{
    "code": "0000",
    "message": "操作成功",
    "data": null
}
```
##### 请求参数说明
节点 | 节点描述 | 取值说明
---|---|---
memberId | 会员id | node端获取
acount | 会员账号 | 手机号
pwd | 交易密码 | 
verificationCode | 短信验证码 | 

##### 返回参数说明
无
