```javascript var config = { type: 'div', url: 'htmlpage.html', title: '标题1' }; _modal = $dialog.modal(config);

    _modal.show();

```

基础属性介绍

type : 内容呈现方式（html，div，iframe）默认html
skin : 皮肤（目前有skin1和skin2，待补充）
template : 重定义模板样式，设置项会覆盖已选skin样式
url : type为ifirame和div时有效
content : 内容，type=html时有效
title : 标题，为空时不显示标题
titlestyle: 标题样式
lock : 是否锁定屏幕，阻塞模式
mask : 是否显示遮罩层
quickClose: 为true时触发快捷关闭对话框功能（esc或点击其他区域）
width :宽度
height :高度
time :关闭时间（毫秒），默认-1，大于0时触发定时关闭效果

常用方法

show()：显示
hide()：隐藏
refresh()： 刷新内容

事件

show
hide
close

自定义模板介绍

配置说明

javascript var template = { alert: { width: '宽度', height: '高度', html: '内容' }, confirm: { width: '宽度', height: '高度', html: '内容' }, tooltip: { width: '宽度', height: '高度', html: '内容' }, modal: { width: '宽度', height: '高度', html: '内容' } };

内容特殊标记

data-dialog-title: 标题
data-dialog-content: 内容
data-dialog-submit: 提交
data-dialog-hide: 隐藏
data-dialog-close: 关闭
data-dialog-drag: 拖动标记
data-dialog-arrows:箭头
