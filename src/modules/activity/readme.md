##活动组件

###版本0.1.4
* 活动组件接入了监控组件,参考test/moni-error下的测试用例
* 只有在线上环境才会发送对应的日志记录,或者测试的url里包含monitor

###版本0.1.3
* 调整了活动接口,去除部分无用的字段
* 不再走跳转加密,接口直接返回加密key
* 新增beforeParse,将filterData转移到beforeParse方法里
* beforeRender里可以处理和返回筛选后的数据

###版本0.1.2

* 修复了同程 app 里后退出现 404 的问题。
* 还有增加了 IsSellOut 的可选参数，用来返回售罄的线路。
* 支持跳转到微信站和普通终页
